## Prologue

코딩 스터디 시즌 1 (2016.12.14 ~ 2017.02.23) 진행을 통해서 각자 스스로의 성향에 대해 이해함.
구성원의 성향에 따라 각자 스스로의 가능성을 최대화 할 수 있는 학습의 필요성을 공감함.
시즌 2는 약 3개월 진행하고 각자 공유할 수 있는 산출물 제작을 목표로 한다.

## Epilogue

2017.06.02 을 마지막으로 스터디 모임 시즌 2가 종료됨.

시즌 3 참석자는 다음 사항에 대해 준비해주길 바람.

1. 신규 repository 이름
2. 시즌 3에서 함께 했으면 하는 사항 3가지
3. 시즌 3에서 각자 해보고 싶은 주제 3가지
4. 시즌 3에서 사용하고자 하는 익명 아이디
